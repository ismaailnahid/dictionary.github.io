# 🚀 Gboard Dictionary Editor Pro

একটি শক্তিশালী ওয়েব-ভিত্তিক টুল যা Gboard ডিকশনারি ফাইল (.txt/.zip) এডিট, ম্যানেজ এবং এক্সপোর্ট করার জন্য তৈরি।

![License](https://img.shields.io/badge/license-MIT-green)
![Platform](https://img.shields.io/badge/platform-Web%20%7C%20Mobile%20%7C%20Desktop-lightgrey)

## ✨ ফিচার সমূহ

### 📁 ফাইল ম্যানেজমেন্ট
- TXT ফাইল আপলোড ও ডাউনলোড
- ZIP ফাইল সাপোর্ট (অটো এক্সট্র্যাক্ট)
- টাইমস্ট্যাম্প সহ ফাইল সেভ (যেমন: `dictionary_02_45_PM_04_April_2026.txt`)

### ✏️ ডিকশনারি এডিটিং
- নতুন এন্ট্রি যোগ (শর্টকাট + ভ্যালু)
- বিদ্যমান এন্ট্রি এডিট ও ডিলিট
- ডুপ্লিকেট এন্ট্রি হাইলাইট ও রিমুভ
- খালি লাইন মুছে ফেলা
- এক্সট্রা স্পেস ক্লিন করা
- A-Z অনুযায়ী সর্টিং

### 🔍 অ্যাডভান্সড ফিল্টারিং
- শর্টকাট ও ভ্যালু অনুযায়ী ফিল্টার
- Contains / Starts With / Ends With অপশন
- শর্টকাট বা ভ্যালু অনুযায়ী সাজানো

### 🎯 ইউজার ইন্টারফেস
- 🌓 ডার্ক/লাইট মোড (অটো ডিটেক্ট)
- 📱 সম্পূর্ণ রেস্পন্সিভ (মোবাইল, ট্যাবলেট, ডেস্কটপ)
- ⌨️ কীবোর্ড শর্টকাট সাপোর্ট
- 🔤 আলফাবেট বার (A-Z জাম্পিং)
- 💫 মসৃণ অ্যানিমেশন ও ট্রানজিশন

### 🚀 অতিরিক্ত ফিচার
- 📤 টেলিগ্রামে ডাইরেক্ট শেয়ার
- 💾 অটো-সেভ ও ব্যাকআপ সিস্টেম
- 📊 লাইভ প্রগ্রেস বার ও স্ট্যাটাস
- 🎨 ফ্লোটিং অ্যাকশন বাটন (FAB)

## ⌨️ কীবোর্ড শর্টকাট

| শর্টকাট | কাজ |
|---------|------|
| `Ctrl+S` | ডাউনলোড (TXT) |
| `Ctrl+D` | ডুপ্লিকেট রিমুভ |
| `Ctrl+E` | খালি লাইন রিমুভ |
| `Ctrl+K` | ফরম্যাট (এক্সট্রা স্পেস ক্লিন) |
| `Ctrl+L` | সর্ট লাইনস |
| `Ctrl+O` | ফাইল ওপেন |
| `Arrow Up/Down` | প্রিভিউতে নেভিগেট |
| `Delete` | সিলেক্টেড এন্ট্রি ডিলিট |
| `Enter` | এন্ট্রি যোগ/আপডেট |

## 🚀 কিভাবে ব্যবহার করবেন

### ধাপ ১: ফাইল আপলোড
1. "Choose File" বাটনে ক্লিক করুন
2. আপনার Gboard এর `dictionary.txt` অথবা ZIP ফাইল সিলেক্ট করুন

### ধাপ ২: ডিকশনারি এডিট
- **নতুন এন্ট্রি যোগ:** শর্টকাট ও ভ্যালু লিখে "Add Entry" ক্লিক করুন
- **এডিট:** প্রিভিউ থেকে যেকোনো এন্ট্রি সিলেক্ট করে, নিচের বক্সে পরিবর্তন করে "Apply" ক্লিক করুন
- **ডিলিট:** এন্ট্রি সিলেক্ট করে "Delete" বাটন ক্লিক করুন

### ধাপ ৩: ফিল্টার ও সার্চ
- শর্টকাট বা ভ্যালু দিয়ে সার্চ করুন
- "Sort By" অপশন ব্যবহার করে সাজান

### ধাপ ৪: ফাইল সেভ
- **TXT ডাউনলোড:** `📥 TXT` বাটন ক্লিক করুন
- **ZIP ডাউনলোড:** `📦 ZIP` বাটন ক্লিক করুন
- ফাইল automatically টাইমস্ট্যাম্প সহ সেভ হবে

### ধাপ ৫: টেলিগ্রামে শেয়ার (অপশনাল)
1. সেটিংস থেকে Bot Token ও Chat ID সেট করুন
2. `📤 Telegram` বাটনে ক্লিক করুন

## 🛠️ ইনস্টলেশন ও রান

### লোকালি রান করতে:
```bash
# রিপোজিটরি ক্লোন করুন
git clone https://github.com/yourusername/gboard-dictionary-editor.git

# প্রকল্পে ঢুকুন
cd gboard-dictionary-editor

# যেকোনো লাইভ সার্ভার ব্যবহার করুন (যেমন: VS Code Live Server, Python HTTP Server)
# Python দিয়ে:
python -m http.server 8000

# বা Node.js দিয়ে:
npx serve .
